# Week 3

## Data can be 

- Primitive
  - Data type given by the compiler
  - Integers, shorts, char

- Composite
  - Combination of primitive types
  - String

Example:

```java
int x = 4; // 4

int[] y = new int[4]; // Stores address, reference.
```
